# README – rattsagt.se

## Innehåll

- **index.html** – Inloggningssida, visas på www.rattsagt.se
- **login.html** – Inloggningsformulär
- **login.js** – Inloggningslogik
- **dashboard.html** – Startsida efter lyckad inloggning
- **gpt-button.js** – Öppnar GPT-chatt från dashboard
- **min-resa.html** – Visar utvecklingen av ert projekt
- **settings.html** – Inställningssida (kommer snart)
- **style.css** – Gemensam stil för alla sidor
- **script.js** – Gemensam scriptfil
- **session-handler.js** – Hanterar sessionslogik
- **logo.svg** – Webbplatsens logotyp

## Användning

1. Ladda upp filerna till GitHub-repot `rattsagt-se`
2. Deploya till Vercel
3. Öppna `https://www.rattsagt.se` – inloggningssidan visas
